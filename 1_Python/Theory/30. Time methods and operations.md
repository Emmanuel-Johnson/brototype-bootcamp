# Time methods and operations

## ‚úÖ 1. Get current date and time

```python
from datetime import datetime

now = datetime.now()
print(now)  # 2025-07-04 23:59:00.123456
```

---

## ‚úÖ 2. Format date/time as string

```python
formatted = now.strftime("%Y-%m-%d %H:%M:%S")
print(formatted)  # '2025-07-04 23:59:00'
```

---

## ‚úÖ 3. Parse string to datetime

```python
dt = datetime.strptime("2025-06-01", "%Y-%m-%d")
print(dt)  # datetime.datetime(2025, 6, 1, 0, 0)
```

---

## ‚úÖ 4. Get today's date

```python
from datetime import date

today = date.today()
print(today)  # 2025-07-04
```

---

## ‚úÖ 5. Get yesterday and tomorrow

```python
from datetime import timedelta

yesterday = today - timedelta(days=1)
tomorrow = today + timedelta(days=1)
```

---

## ‚úÖ 6. Difference between two dates/times

```python
start = datetime(2025, 6, 1)
end = datetime(2025, 6, 15)
diff = end - start
print(diff.days)  # 14
```

---

## ‚úÖ 7. Add days/months/hours to a date

```python
future = now + timedelta(days=10, hours=5)
```

---

## ‚úÖ 8. Extract year, month, day, weekday

```python
print(now.year)   # 2025
print(now.month)  # 7
print(now.day)    # 4
print(now.weekday())  # 3 (0 = Monday, 6 = Sunday)
```

---

## ‚úÖ 9. Convert datetime to timestamp and back

```python
timestamp = now.timestamp()
print(timestamp)  # 1757051940.123456

dt_from_ts = datetime.fromtimestamp(timestamp)
```

---

## ‚úÖ 10. Check if date is weekend

```python
def is_weekend(d):
    return d.weekday() >= 5  # 5 = Saturday, 6 = Sunday

print(is_weekend(date(2025, 7, 6)))  # True (Sunday)
```

---

## ‚úÖ 11. Timezone-aware datetime (basic)

```python
from datetime import timezone

aware = datetime.now(timezone.utc)
```

> üß† For advanced timezone handling, use `pytz` or `zoneinfo` (Python 3.9+).

---

## ‚úÖ 12. Measure execution time (interview trick)

```python
import time

start = time.time()
# some_code()
end = time.time()
print(f"Execution time: {end - start:.2f} seconds")
```

---

## ‚úÖ 13. Sleep / Wait for some seconds

```python
import time

time.sleep(2)  # wait for 2 seconds
```

## ‚ö†Ô∏è Common Interview Questions

| Question                                      | Feature Used                  |
|-----------------------------------------------|------------------------------|
| How to get yesterday's date?                  | `timedelta(days=1)`          |
| Convert `"2024-01-01"` to datetime            | `strptime()`                 |
| How many days between two dates?              | `end - start`                |
| How to add 1 hour to current time?            | `timedelta(hours=1)`         |
| Print current date in format "DD-MM-YYYY"     | `strftime("%d-%m-%Y")`       |
| Get the day of the week from a date           | `weekday()` or `strftime("%A")` |
